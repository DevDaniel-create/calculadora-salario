# Calculadora de Salário

Aplicação web para calcular o salário de funcionários com acréscimo de 10%.

## 🔗 Acesse o app
[Clique aqui para acessar](https://calculadora-salario-salario-app-8lmjfs.streamlit.app/)

## 💻 Tecnologias utilizadas
- Python
- Streamlit

## ⚙️ Como funciona
1. Informe o valor da hora do funcionário
2. Informe a quantidade de horas trabalhadas no mês
3. Clique em **Calcular**
4. O app exibe o salário bruto, o acréscimo de 10% e o total a pagar
